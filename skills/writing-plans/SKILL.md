# Skill: Writing Implementation Plans

## 📋 Objectif
Décomposer une spécification complexe en petits morceaux digestibles par un agent pour garantir le succès global du projet.

## 🛠️ Processus
1. **Analyser** : Regarder la `SPECIFICATION.md` validée.
2. **Découper** : Créer 3 à 8 tâches "atomiques" (environ 5 minutes de code chacune).
3. **Structurer** : Chaque tâche doit avoir :
    - Un titre clair.
    - Sa propre vérification (Test critère).
    - Les fichiers impactés.

## 📝 Format de sortie
L'agent doit produire un document `IMPLEMENTATION_PLAN.md` contenant :
- Contexte général du projet.
- Tableau des tâches (ID, Statut, Description, Fichiers, Test).
