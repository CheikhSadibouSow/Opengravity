# Skill: n8n Workflow Automation

## 💡 Objectif
Donner à l'agent la capacité de concevoir des workflows complexes sur n8n, de les exporter et de gérer les webhooks.

## 🛠️ Processus
1. **Définer le Trigger** : Webhook, Cron, ou événement API.
2. **Mapper les Données** : Transformer les JSON d'entrée en sorties utilisables.
3. **Logique de Nœuds** : Utiliser des nœuds 'Function' (Javascript) ou des nœuds natifs de n8n.
4. **Gestion d'Erreurs** : Mettre en place des mécanismes de 'Error Handling' (nœuds On Error).

## 📝 Format de sortie
- Schéma JSON du workflow.
- Instructions détaillées d'importation.
- URL de webhook suggérée (pour les webhooks entrants).
