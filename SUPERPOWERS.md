# 🦸‍♂️ OpenGravity Superpowers

Votre agent dispose désormais de capacités d'ingénierie avancées. À partir de maintenant, quand vous demanderez de créer une nouvelle application ou une tâche complexe, l'agent suivra ce flux :

## 1. 🧠 Brainstorming (Design Thinking)
Je ne code plus à l'aveugle. Je vais d'abord vous poser des questions pour comprendre votre vision. Résultat : une boîte de dialogue et un fichier `SPECIFICATION.md`.

## 2. 📝 Implementation Planning
Une fois la spécification validée, je vais diviser tout le travail en petites tâches précises dans `IMPLEMENTATION_PLAN.md`.

## 3. 🧪 TDD (Test-Driven Development)
Avant d'écrire le code final, je prépare les tests (si possible) pour m'assurer que les fondations sont solides.
